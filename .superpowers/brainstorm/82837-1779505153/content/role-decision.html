How Each Layer Is Identified
Layer role value Access Scope Routes
Superadmin "superadmin" All tenants (God Mode) /superadmin/*
Senior "senior" Own tenant_id only /senior/*
User "user" Own tenant_id + RBAC actions /user/*
Frontend middleware logic:
if (role === "superadmin") → allow /superadmin/*
if (role === "senior")    → allow /senior/*
if (role === "user")      → allow /user/*
else → redirect to / (login)
