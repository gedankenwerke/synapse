Design Summary — Tenant-Based User Management

สรุป design ที่เลือกทั้งหมด

✅ การเลือกทั้งหมด

1

Tenant Selection → Card Grid

แสดง tenant เป็น card มีชื่อ + จำนวน user + parent tenant

2

View Details → Modal กลางจอ

กด 👁 แสดง Modal ดูรายละเอียด user (ชื่อ, tenant, role, permissions)

3

Edit User → Modal แก้ไข

กด ✏️ เปิด Modal ฟอร์มแก้ไขข้อมูล user ทั้งหมด

4

Delete User → Confirmation Modal

กด 🗑️ เปิด Modal ยืนยันการลบ แสดงชื่อ user ที่จะลบ
